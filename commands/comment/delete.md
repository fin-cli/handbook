# fin comment delete

Deletes a comment.

### OPTIONS

&lt;id&gt;...
: One or more IDs of comments to delete.

[\--force]
: Skip the trash bin.

### EXAMPLES

    # Delete comment.
    $ fin comment delete 1337 --force
    Success: Deleted comment 1337.

    # Delete multiple comments.
    $ fin comment delete 1337 2341 --force
    Success: Deleted comment 1337.
    Success: Deleted comment 2341.

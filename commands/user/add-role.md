# wp user add-role

Adds a role for a user.

### OPTIONS

&lt;user&gt;
: User ID, user email, or user login.

&lt;role&gt;
: Add the specified role to the user.

### EXAMPLES

    $ wp user add-role 12 author
    Success: Added 'author' role for johndoe (12).

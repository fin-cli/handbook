# Then /^STDOUT should be YAML containing:$/

Expect STDOUT to be YAML containig certain content.


***

## Usage

```
Scenario: My example scenario
  When I run `fin cli alias list`
  Then STDOUT should be YAML containing:
    """
    @all: Run command against every registered alias.
    @foo:
      path: {TEST_DIR}/foo
    """
```
